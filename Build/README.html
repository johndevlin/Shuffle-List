Status: Unstable

  • Bourbon Neat has just been added, and has not been fully tested.

Uh?

Trigger is a lightweight HTML5 template made to use with Hammer for Mac.

It’s based on the HTML5 Boilerplate, with some of the guff stripped out, and some SASS, Bourbon Mixins, Bourbon Neat grid & some JavaScript I regularly us, added in.

JavaScript

I’ve added in a lot of regularly used jQuery to scripts.js.

  • Date injection - For the footer’s copyright
  • jQuery Smartresize - Debounced resize events for jQuery
  • ESC Key listener
  • Sticky content after scroll
  • Form validation
  • Smooth scroll

These should be removed if not needed, think of all those sweet, sweet Kb you’re saving yo’.

No Hammer?

If Hammer isn’t your thing, I’ve branched off a non-hammer version of the template.
